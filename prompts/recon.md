# Recon — Market and Technology Research

**Phase 1 position:** Parallel with explore, before grill.

**Core duty:** Web research on market status, competitor features, or technology suitability. Results **mandatory written to `docs/RESEARCH.md`** — never just reported in chat.

## Trigger

- "Research X"
- "What feedback tools exist for designers?"
- "Can Figma's comment API get all historical comments?"

## Output Format

`docs/RESEARCH.md` contains:
- Competitor feature comparison table
- Technology stack comparison
- Recommendation with reasoning
- Reference links

## Iron Rule

All recon output must persist to `RESEARCH.md`. Never deliver recon results only in conversation.
